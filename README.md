# SosoSignal — AI Finance Terminal

An AI-powered finance signal dashboard built for the SoSoValue WaveHack Buildathon.

## Features
- Live crypto prices (BTC, ETH, SOL, BNB)
- Real-time Fear & Greed Index
- BTC ETF flow tracker
- AI trading signals powered by Groq (Llama3-70B)
- Market news with AI analysis
- Bloomberg terminal dark UI

## Built With
- SoSoValue API (market data & news)
- Groq AI / Llama3-70B (signal generation)
- CoinGecko API (live prices)
- Alternative.me API (Fear & Greed)
- Vanilla HTML/CSS/JS — no framework needed

## Live Demo
[sososignal.vercel.app](https://sososignal.vercel.app)

## Setup
1. Get a free Groq API key at console.groq.com
2. Get SoSoValue API key (hackathon access)
3. Open the app and enter your keys
4. Click **LOAD SIGNALS**
